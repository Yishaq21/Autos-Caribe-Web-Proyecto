# Autos Caribe Web

## Descripción

Autos Caribe Web es una aplicación desarrollada para la gestión de inventario y venta de vehículos de la empresa Autos Caribe

## Objetivo

Facilitar la publicación, búsqueda y consulta de vehículos mediante una plataforma web centralizada.

## Tecnologías Previstas

- Java
- Spring Boot
- Bootstrap
- MySQL
